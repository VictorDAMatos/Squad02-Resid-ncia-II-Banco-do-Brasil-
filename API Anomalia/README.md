# 🏦 API Banco do Brasil - Detecção de Fraudes (Squad 02)

Esta é uma API desenvolvida em **FastAPI** com banco de dados **SQLite**, projetada para registrar transações bancárias, filtrá-las e detectar possíveis anomalias/fraudes utilizando regras de negócio específicas. O projeto também conta com um módulo bônus de atendimento via IA.

## 🚀 Tecnologias Utilizadas
- **Python 3.10+**
- **FastAPI** (Framework da API)
- **Uvicorn** (Servidor ASGI)
- **SQLite3** (Banco de Dados Relacional)
- **Scalar** (Documentação Visual)
